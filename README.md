# 品牌分析系统 (Brand Analysis Platform)

## 项目概述

这是一个全自动化的品牌分析平台，能够：
- 🔍 **全网数据采集**：自动爬取品牌在各平台（小红书、抖音、微博、知乎、B站等）的信息
- 🤖 **AI智能分析**：使用大语言模型对数据进行深度分析
- 📊 **专业报告生成**：自动生成类似咨询公司的品牌分析报告
- 🖥️ **可视化平台**：提供Web界面进行品牌分析和报告查看

## 核心功能模块

### 1. 数据采集模块
- **多平台支持**：集成 MediaCrawler，支持小红书、抖音、微博、知乎、B站等
- **关键词搜索**：根据品牌名称自动搜索相关内容
- **定时任务**：支持定时自动采集，保持数据更新
- **数据清洗**：去重、去噪、格式化处理

### 2. AI分析模块
- **情感分析**：分析用户对品牌的情感倾向（正面/负面/中性）
- **主题提取**：识别讨论的主要话题和关注点
- **关键词分析**：提取高频词汇，分析品牌形象
- **趋势分析**：分析品牌热度变化趋势
- **竞品对比**：对比分析竞品表现
- **深度洞察**：使用大语言模型生成深度分析

### 3. 报告生成模块
- **报告模板**：专业的品牌分析报告模板
- **数据可视化**：自动生成图表和仪表盘
- **多格式导出**：支持 PDF、Word、Excel 等格式
- **自定义报告**：支持用户自定义报告内容

### 4. Web平台界面
- **品牌管理**：添加、管理要分析的品牌
- **任务监控**：实时查看数据采集和分析任务状态
- **数据看板**：可视化展示分析结果
- **报告查看**：在线查看和下载分析报告

## 技术栈

### 后端
- **Python 3.11+**
- **FastAPI** - Web框架和API服务
- **MediaCrawler** - 多平台数据爬虫
- **Celery** - 异步任务队列
- **SQLAlchemy** - ORM框架
- **Pandas** - 数据处理
- **OpenAI API / 本地LLM** - AI分析
- **Jinja2** - 报告模板引擎

### 前端
- **React / Vue.js** - 前端框架
- **ECharts / D3.js** - 数据可视化
- **Ant Design / Element UI** - UI组件库

### 数据库
- **MySQL** - 结构化数据存储
- **MongoDB** - 非结构化数据存储
- **Redis** - 缓存和任务队列

## 实现可行性分析

✅ **完全可行！** 这个系统可以搭建，原因如下：

1. **数据采集**：MediaCrawler 已经支持多平台爬取
2. **AI分析**：现有的大语言模型API（OpenAI、Claude、本地模型）可以完成分析任务
3. **报告生成**：模板引擎 + 数据填充可以生成专业报告

## 使用示例

```python
# 1. 添加品牌分析任务
POST /api/brands
{
  "brand_name": "山四砂锅",
  "platforms": ["xhs", "douyin", "weibo", "zhihu"],
  "analysis_type": "full"
}

# 2. 启动数据采集
POST /api/tasks/crawl
{
  "brand_id": 1,
  "keywords": ["山四砂锅", "山四砂锅品牌"]
}

# 3. 启动AI分析
POST /api/tasks/analyze
{
  "brand_id": 1
}

# 4. 生成报告
POST /api/reports/generate
{
  "brand_id": 1,
  "format": "pdf"
}
```
